# Kauz

Kauz is an AI-powered founder network.

## Core Features

- Authentication
- Founder Profiles
- Social Feed
- Founder Discovery
- AI Co-Founder Chat
- AI Twin Training
- Startup Command Center

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Firebase Authentication
- Firestore Database
- Gemini API

## Vision

Build the world's first AI-powered operating system for founders.
